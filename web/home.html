Booking Scrapper API

V1.0

You can use this API to obtain relevant information of whatever hotel you are interested in. In the box bellow, you can see the paths you need to use according to reach your purpose. Any help you may need is also included after this paragraph.

  • To see all hotels you have looked for in this session go to http://localhost:8086/v1/hotels
  • To get the information of a concrete hotel, http://localhost:8086/v1/hotels/:id?pages=100, where id is the name booking gives the hotel in its url. The query param pages is the amount of comment pages to extract from booking and it only works if the hotel has not been stored to de datamart.
  • To check reviews of any hotel, whether you have searched it before or not, go to http://localhost:8086/v1/hotels/:id/comments
  • To check services of any hotel go to http://localhost:8086/v1/hotels/:id/services
  • To check ratings for different offered services of any hotel go to http://localhost:8086/v1/hotels/:id/ratings

Because of the way this API has been implemented, whenever you look for a hotel it will automatically be saved for your user. This way consults for any of these hotels will be faster, since the information has been locally stored to avoid sending repetitive requests to Booking servers.
